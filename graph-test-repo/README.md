# Graph Test Repository

This repository is intentionally structured to produce a clean and expressive dependency graph.

## Clusters

- `app/api/routes`: Entry points
- `app/services`: Business orchestration
- `app/repositories`: Data access layer
- `app/models`: Domain entities
- `app/core`: Shared infra
- `app/metrics`: Telemetry
- `app/workers`: Scheduled jobs

Use this folder as test input for ingestion.
