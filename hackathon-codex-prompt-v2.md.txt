## WHY THESE TECH CHOICES (Read This First)

**Backend → Python FastAPI** (not Node/Express)
Reason: All the heavy lifting here is Python-native — AST parsing, embedding libraries,
ChromaDB, LangChain if needed. FastAPI gives async support, automatic Swagger docs,
and is faster to write than Express for AI/ML pipelines. Do NOT use Node.js for backend.

**Vector DB → ChromaDB** (not Supabase/pgvector)
Reason: ChromaDB runs fully in-process with zero config. No cloud account needed.
Persists to disk automatically. Has a Python client that is 3 lines to set up.
Perfect for hackathon — works offline, no rate limits, instant.

**Background Jobs → Inngest**
Reason: Embedding a whole repo takes 30-120 seconds. Without Inngest, the HTTP request
would time out. Inngest lets us fire-and-forget the heavy work, stream progress back
to the frontend via events, and retry on failure automatically.

**Frontend → React + Vite + Framer Motion + TailwindCSS**
Reason: Framer Motion is the gold standard for React animations. Combined with
ReactFlow for the graph and a dark hacker aesthetic, this will look like a
production tool, not a hackathon project.

## PROJECT OVERVIEW

Build **CodeLens AI** — a full-stack platform that gives any developer an X-ray
of any unfamiliar codebase in under 60 seconds.

### The 5 Modules:
1. **Codebase Ingestion** — ZIP upload or GitHub URL → parse → embed → store in ChromaDB
2. **Interactive Dependency Map** — Animated graph of files, functions, classes, imports
3. **Semantic Error Diagnosis** — Paste error → semantic search → find culprit files
4. **Closed-Loop Auto-Fix Sandbox** — Clone → run → detect error → GPT fix → retry loop
5. **Chat with Codebase** — RAG chat over the indexed codebase

### Inngest powers the async backbone:
- Ingestion job (long running) → Inngest function
- Embedding pipeline → Inngest step functions with retries
- Sandbox loop → Inngest function with iteration steps

## STEP 2 — CHROMADB SETUP (`backend/services/chroma_client.py`)

ChromaDB needs ZERO external setup. It runs in-process and saves to disk.

```python
import chromadb
from chromadb.config import Settings
import os

# This is all the setup needed — ChromaDB handles everything else
client = chromadb.PersistentClient(
    path=os.getenv("CHROMA_PERSIST_DIR", "./chroma_data")
)

def get_collection(project_id: str):
    """Get or create a collection for a project. Each project = one collection."""
    return client.get_or_create_collection(
        name=f"project_{project_id}",
        metadata={"hnsw:space": "cosine"}   # Use cosine similarity
    )

def add_chunks(project_id: str, chunks: list[dict], embeddings: list[list[float]]):
    """
    Store chunks with their embeddings.
    chunks = [{ id, code_text, file_path, chunk_name, chunk_type, metadata }]
    embeddings = list of 1536-dim float vectors
    """
    collection = get_collection(project_id)
    collection.add(
        ids=[c["id"] for c in chunks],
        embeddings=embeddings,
        documents=[c["code_text"] for c in chunks],
        metadatas=[{
            "file_path": c["file_path"],
            "chunk_name": c["chunk_name"],
            "chunk_type": c["chunk_type"],
            "line_start": c["metadata"].get("line_start", 0),
            "line_end": c["metadata"].get("line_end", 0),
            "dependencies": str(c["metadata"].get("dependencies", [])),
            "language": c["metadata"].get("language", "unknown")
        } for c in chunks]
    )

def query_similar(project_id: str, query_embedding: list[float], n_results: int = 10):
    """Find most similar chunks to a query embedding."""
    collection = get_collection(project_id)
    results = collection.query(
        query_embeddings=[query_embedding],
        n_results=n_results,
        include=["documents", "metadatas", "distances"]
    )
    return results

def get_all_chunks(project_id: str):
    """Retrieve all chunks for building the dependency graph."""
    collection = get_collection(project_id)
    return collection.get(include=["documents", "metadatas"])

def delete_project(project_id: str):
    """Clean up a project's data."""
    client.delete_collection(f"project_{project_id}")
```

**Key facts about ChromaDB for Codex to understand:**
- No Docker, no cloud, no API key needed
- Data persists in `./chroma_data/` folder between restarts
- Each project gets its own named collection
- Similarity search returns distances (0=identical, 2=opposite) — convert to score = 1 - distance

```python
import inngest
import os

# Initialize the Inngest client
inngest_client = inngest.Inngest(
    app_id="codelens-ai",
    event_key=os.getenv("INNGEST_EVENT_KEY"),
    signing_key=os.getenv("INNGEST_SIGNING_KEY"),

# ─────────────────────────────────────────────────────────
# INNGEST FUNCTION 1: Full ingestion + embedding pipeline
# ─────────────────────────────────────────────────────────
@inngest_client.create_function(
    fn_id="ingest-codebase",
    trigger=inngest.TriggerEvent(event="codelens/ingest.requested"),
    retries=2
)
async def ingest_codebase(ctx: inngest.Context, step: inngest.Step):
    data = ctx.event.data
    project_id = data["project_id"]
    source_path = data["source_path"]   # local path after upload/clone

    # Step 1: Parse all files into chunks
    await step.send_event("update-status", {
        "name": "codelens/status.updated",
        "data": {"project_id": project_id, "status": "parsing", "progress": 10}
    })
    chunks = await step.run("parse-files", lambda: parse_all_files(source_path, project_id))

    # Step 2: Generate embeddings in batches
    await step.send_event("update-status", {
        "name": "codelens/status.updated",
        "data": {"project_id": project_id, "status": "embedding", "progress": 40}
    })
    embeddings = await step.run("generate-embeddings", lambda: generate_all_embeddings(chunks))

    # Step 3: Store in ChromaDB
    await step.send_event("update-status", {
        "name": "codelens/status.updated",
        "data": {"project_id": project_id, "status": "storing", "progress": 80}
    })
    await step.run("store-in-chroma", lambda: store_chunks(project_id, chunks, embeddings))

    # Step 4: Build dependency graph
    graph = await step.run("build-graph", lambda: build_dependency_graph(project_id, chunks))

    # Done
    await step.send_event("complete", {
        "name": "codelens/status.updated",
        "data": {
            "project_id": project_id,
            "status": "complete",
            "progress": 100,
            "chunk_count": len(chunks),
            "graph": graph
        }
    })
    return {"status": "complete", "chunks": len(chunks)}


# ─────────────────────────────────────────────────────────
# INNGEST FUNCTION 2: Closed-loop auto-fix sandbox
# ─────────────────────────────────────────────────────────
@inngest_client.create_function(
    fn_id="sandbox-autofix",
    trigger=inngest.TriggerEvent(event="codelens/sandbox.run"),
    retries=1,
    timeout="10m"
)
async def sandbox_autofix(ctx: inngest.Context, step: inngest.Step):
    data = ctx.event.data
    project_id = data["project_id"]
    repo_url = data["repo_url"]
    run_command = data["run_command"]
    target_file = data["target_file"]
    max_iterations = 3
    iterations = []

    # Clone the repo
    sandbox_path = await step.run("clone-repo", lambda: clone_repo(repo_url, project_id))

    for i in range(max_iterations):
        # Run the command
        result = await step.run(f"run-iteration-{i}", lambda: run_command_in_sandbox(sandbox_path, run_command))

        if result["success"]:
            await step.send_event("sandbox-success", {
                "name": "codelens/sandbox.updated",
                "data": {"project_id": project_id, "status": "success", "iterations": iterations}
            })
            return {"success": True, "iterations": iterations}

        # Generate fix
        fix = await step.run(f"generate-fix-{i}", lambda: generate_fix_with_gpt(
            sandbox_path, target_file, result["error"]
        ))
        iterations.append({"iteration": i+1, "error": result["error"], "fix": fix["summary"], "diff": fix["diff"]})

        # Apply fix
        await step.run(f"apply-fix-{i}", lambda: apply_fix_to_file(sandbox_path, target_file, fix["code"]))

        # Emit progress
        await step.send_event(f"sandbox-progress-{i}", {
            "name": "codelens/sandbox.updated",
            "data": {"project_id": project_id, "status": "iterating", "iteration": i+1, "iterations": iterations}
        })

    # Failed after max iterations
    return {"success": False, "iterations": iterations}
```

## MODULE 1: CODEBASE INGESTION (`backend/routes/ingest.py`)

```python
from fastapi import APIRouter, UploadFile, File, Form, BackgroundTasks
from services.parser import parse_all_files
import uuid, os, zipfile, shutil
from git import Repo
from inngest_client import inngest_client
import aiofiles

router = APIRouter()

# In-memory project status store (use Redis in production)
project_status = {}

@router.post("/ingest")
async def ingest_codebase(
    mode: str = Form(...),           # "upload" or "github"
    github_url: str = Form(None),
    branch: str = Form("main"),
    file: UploadFile = File(None)
):
    project_id = str(uuid.uuid4())[:8]
    source_path = f"/tmp/codelens/{project_id}"
    os.makedirs(source_path, exist_ok=True)

    if mode == "upload" and file:
        # Save and extract ZIP
        zip_path = f"{source_path}/upload.zip"
        async with aiofiles.open(zip_path, "wb") as f:
            content = await file.read()
            await f.write(content)
        with zipfile.ZipFile(zip_path, "r") as z:
            z.extractall(source_path)
        os.remove(zip_path)

    elif mode == "github" and github_url:
        # Clone with depth=1 for speed
        token = os.getenv("GITHUB_TOKEN", "")
        if token:
            # Inject token for private repos
            auth_url = github_url.replace("https://", f"https://{token}@")
        else:
            auth_url = github_url
        Repo.clone_from(auth_url, source_path, depth=1, branch=branch)

    else:
        return {"error": "Provide either a ZIP file or a GitHub URL"}

    # Fire the Inngest event — this starts the background job
    await inngest_client.send(inngest.Event(
        name="codelens/ingest.requested",
        data={"project_id": project_id, "source_path": source_path}
    ))

    # Save initial status
    project_status[project_id] = {"status": "queued", "progress": 0}

    return {
        "project_id": project_id,
        "status": "queued",
        "message": "Ingestion started. Poll /api/status/{project_id} for progress."
    }


@router.get("/status/{project_id}")
async def get_status(project_id: str):
    """Frontend polls this every 2 seconds to show progress."""
    status = project_status.get(project_id, {"status": "not_found"})
    return status
```

## MODULE 2: PARSER (`backend/services/parser.py`)

```python
"""
Parse source files into semantic chunks: functions, classes, imports.
Each chunk becomes one embedding in ChromaDB.
"""
import os, glob, re, uuid
from pathlib import Path

IGNORED_DIRS = {"node_modules", ".git", "dist", "build", "__pycache__", ".next", "venv", ".venv"}
CODE_EXTENSIONS = {".js", ".ts", ".jsx", ".tsx", ".py", ".java", ".go", ".rb", ".cpp", ".c", ".rs", ".vue", ".php"}

def parse_all_files(source_path: str, project_id: str) -> list[dict]:
    """Walk all files and parse them into chunks."""
    all_chunks = []
    files = find_code_files(source_path)

    for file_path in files:
        try:
            with open(file_path, "r", encoding="utf-8", errors="ignore") as f:
                content = f.read()
            ext = Path(file_path).suffix
            relative_path = os.path.relpath(file_path, source_path)

            if ext in {".js", ".ts", ".jsx", ".tsx"}:
                chunks = parse_javascript(content, relative_path)
            elif ext == ".py":
                chunks = parse_python(content, relative_path)
            else:
                chunks = parse_generic(content, relative_path, ext)

            # Add unique IDs
            for chunk in chunks:
                chunk["id"] = f"{project_id}_{str(uuid.uuid4())[:8]}"
                chunk["metadata"]["language"] = ext.lstrip(".")
            all_chunks.extend(chunks)
        except Exception as e:
            print(f"Could not parse {file_path}: {e}")
            continue

    return all_chunks

def find_code_files(source_path: str) -> list[str]:
    files = []
    for root, dirs, filenames in os.walk(source_path):
        # Skip ignored directories
        dirs[:] = [d for d in dirs if d not in IGNORED_DIRS]
        for fname in filenames:
            if Path(fname).suffix in CODE_EXTENSIONS:
                files.append(os.path.join(root, fname))
    return files

def parse_javascript(content: str, file_path: str) -> list[dict]:
    """Extract functions, classes, and imports from JS/TS using regex."""
    chunks = []
    lines = content.split("\n")

    # Extract imports
    imports = []
    import_pattern = re.compile(r"^import\s+.+\s+from\s+['\"](.+)['\"]", re.MULTILINE)
    require_pattern = re.compile(r"require\(['\"](.+)['\"]\)")
    imports = import_pattern.findall(content) + require_pattern.findall(content)

    # Extract named functions: function foo() { ... }
    func_pattern = re.compile(r"(?:export\s+)?(?:async\s+)?function\s+(\w+)\s*\(", re.MULTILINE)
    # Extract arrow functions: const foo = () => { ... } or const foo = async () => ...
    arrow_pattern = re.compile(r"(?:export\s+)?const\s+(\w+)\s*=\s*(?:async\s+)?\(", re.MULTILINE)
    # Extract classes: class Foo {
    class_pattern = re.compile(r"(?:export\s+)?class\s+(\w+)", re.MULTILINE)

    # Combine all named items — extract 30-line windows around each
    for pattern, chunk_type in [(func_pattern, "function"), (arrow_pattern, "function"), (class_pattern, "class")]:
        for match in pattern.finditer(content):
            name = match.group(1)
            start_line = content[:match.start()].count("\n")
            end_line = min(start_line + 40, len(lines))
            chunk_text = "\n".join(lines[start_line:end_line])
            chunks.append({
                "file_path": file_path,
                "chunk_name": name,
                "chunk_type": chunk_type,
                "code_text": chunk_text,
                "metadata": {
                    "line_start": start_line,
                    "line_end": end_line,
                    "dependencies": imports
                }
            })

    # Always add a file-level chunk with imports for graph building
    chunks.append({
        "file_path": file_path,
        "chunk_name": file_path,
        "chunk_type": "file",
        "code_text": content[:500],   # First 500 chars as summary
        "metadata": {"line_start": 0, "line_end": len(lines), "dependencies": imports}
    })
    return chunks

def parse_python(content: str, file_path: str) -> list[dict]:
    """Extract functions and classes from Python using regex."""
    chunks = []
    lines = content.split("\n")
    imports = re.findall(r"^(?:import|from)\s+([\w.]+)", content, re.MULTILINE)

    func_pattern = re.compile(r"^(?:async\s+)?def\s+(\w+)\s*\(", re.MULTILINE)
    class_pattern = re.compile(r"^class\s+(\w+)", re.MULTILINE)

    for pattern, chunk_type in [(func_pattern, "function"), (class_pattern, "class")]:
        for match in pattern.finditer(content):
            name = match.group(1)
            start_line = content[:match.start()].count("\n")
            end_line = min(start_line + 40, len(lines))
            chunk_text = "\n".join(lines[start_line:end_line])
            chunks.append({
                "file_path": file_path,
                "chunk_name": name,
                "chunk_type": chunk_type,
                "code_text": chunk_text,
                "metadata": {"line_start": start_line, "line_end": end_line, "dependencies": imports}
            })

    chunks.append({
        "file_path": file_path, "chunk_name": file_path, "chunk_type": "file",
        "code_text": content[:500],
        "metadata": {"line_start": 0, "line_end": len(lines), "dependencies": imports}
    })
    return chunks

def parse_generic(content: str, file_path: str, ext: str) -> list[dict]:
    """For unknown languages: chunk by 50-line sliding windows."""
    chunks = []
    lines = content.split("\n")
    for i in range(0, len(lines), 40):
        window = "\n".join(lines[i:i+50])
        chunks.append({
            "file_path": file_path,
            "chunk_name": f"{file_path}:L{i}",
            "chunk_type": "block",
            "code_text": window,
            "metadata": {"line_start": i, "line_end": i+50, "dependencies": []}
        })
    return chunks
```

## MODULE 3: EMBEDDING PIPELINE (`backend/services/embedder.py`)

```python
"""
Convert code chunks to OpenAI embeddings.
Always batch — never call one-by-one.
"""
import openai, os, asyncio
from openai import AsyncOpenAI

client = AsyncOpenAI(api_key=os.getenv("OPENAI_API_KEY"))

async def generate_all_embeddings(chunks: list[dict]) -> list[list[float]]:
    """
    Takes all chunks, returns list of embedding vectors in same order.
    Batches 100 at a time to stay within OpenAI limits.
    """
    texts = [c["code_text"][:8000] for c in chunks]   # Truncate to 8k chars max
    all_embeddings = []
    batch_size = 100

    for i in range(0, len(texts), batch_size):
        batch = texts[i:i+batch_size]
        response = await client.embeddings.create(
            model="text-embedding-3-small",
            input=batch
        )
        batch_embeddings = [item.embedding for item in response.data]
        all_embeddings.extend(batch_embeddings)

        # Rate limit buffer between batches
        if i + batch_size < len(texts):
            await asyncio.sleep(0.3)

    return all_embeddings

async def embed_single(text: str) -> list[float]:
    """Embed a single query (for error diagnosis and chat)."""
    response = await client.embeddings.create(
        model="text-embedding-3-small",
        input=text[:8000]
    )
    return response.data[0].embedding
```

```python
from fastapi import APIRouter
from services.chroma_client import get_all_chunks
from collections import defaultdict
import ast, re

router = APIRouter()

@router.get("/map/{project_id}")
async def get_dependency_map(project_id: str):
    """
    Returns nodes and edges for the React Flow graph.
    Nodes = unique files
    Edges = import relationships between files
    """
    result = get_all_chunks(project_id)
    metadatas = result["metadatas"]
    documents = result["documents"]

    files = {}     # file_path → { functions, classes, imports }
    edges = []
    edge_set = set()

    for meta in metadatas:
        fp = meta["file_path"]
        if fp not in files:
            files[fp] = {"functions": [], "classes": [], "imports": [], "chunk_count": 0}
        files[fp]["chunk_count"] += 1

        if meta["chunk_type"] == "function":
            files[fp]["functions"].append(meta["chunk_name"])
        elif meta["chunk_type"] == "class":
            files[fp]["classes"].append(meta["chunk_name"])
        elif meta["chunk_type"] == "file":
            # Parse dependency strings back into list
            deps_str = meta.get("dependencies", "[]")
            try:
                deps = ast.literal_eval(deps_str)
            except:
                deps = []
            files[fp]["imports"] = deps

            # Create edges for each import
            for dep in deps:
                # Try to resolve relative imports to actual file paths
                resolved = resolve_import(dep, fp, list(files.keys()))
                if resolved and (fp, resolved) not in edge_set:
                    edges.append({"source": fp, "target": resolved, "type": "imports"})
                    edge_set.add((fp, resolved))

    # Build nodes
    nodes = []
    for fp, data in files.items():
        file_name = fp.split("/")[-1]
        nodes.append({
            "id": fp,
            "data": {
                "label": file_name,
                "full_path": fp,
                "functions": data["functions"],
                "classes": data["classes"],
                "imports": data["imports"],
                "chunk_count": data["chunk_count"],
                "is_entry": file_name in ["index.js", "main.py", "app.py", "index.ts", "main.ts"]
            },
            "type": "codeNode"   # Custom ReactFlow node type
        })

    stats = {
        "total_files": len(files),
        "total_functions": sum(len(f["functions"]) for f in files.values()),
        "total_classes": sum(len(f["classes"]) for f in files.values()),
        "total_edges": len(edges)
    }
    return {"nodes": nodes, "edges": edges, "stats": stats}

def resolve_import(import_path: str, current_file: str, all_files: list[str]) -> str | None:
    """Try to match an import string to an actual file in the project."""
    if import_path.startswith("."):
        current_dir = "/".join(current_file.split("/")[:-1])
        candidates = [
            f"{current_dir}/{import_path.lstrip('./')}",
            f"{current_dir}/{import_path.lstrip('./')}.js",
            f"{current_dir}/{import_path.lstrip('./')}.ts",
            f"{current_dir}/{import_path.lstrip('./')}/index.js",
        ]
        for c in candidates:
            if c in all_files:
                return c
    return None
```

## MODULE 5: SEMANTIC ERROR DIAGNOSIS (`backend/routes/diagnose.py`)

```python
from fastapi import APIRouter
from pydantic import BaseModel
from services.embedder import embed_single
from services.chroma_client import query_similar
import re

router = APIRouter()

class DiagnoseRequest(BaseModel):
    project_id: str
    error_text: str

@router.post("/diagnose")
async def diagnose_error(req: DiagnoseRequest):
    # 1. Embed the error text
    error_embedding = await embed_single(req.error_text)

    # 2. Semantic search in ChromaDB
    results = query_similar(req.project_id, error_embedding, n_results=15)

    # 3. Extract file paths from stack trace (for score boosting)
    stack_files = re.findall(r"(?:at .+? \(|File \")(.+?)(?::\d+|\", line)", req.error_text)
    stack_files = [f.split("/")[-1] for f in stack_files]   # Just filenames

    # 4. Build ranked results with boosting
    ranked = []
    ids = results["ids"][0]
    docs = results["documents"][0]
    metas = results["metadatas"][0]
    distances = results["distances"][0]

    seen_files = set()
    for i, (doc, meta, dist) in enumerate(zip(docs, metas, distances)):
        similarity = round(1 - dist, 3)
        file_name = meta["file_path"].split("/")[-1]

        # Boost score if file appears in stack trace
        if file_name in stack_files:
            similarity = min(1.0, similarity + 0.25)

        # Deduplicate by file — show best chunk per file
        file_key = meta["file_path"]
        if file_key in seen_files:
            continue
        seen_files.add(file_key)

        ranked.append({
            "file_path": meta["file_path"],
            "chunk_name": meta["chunk_name"],
            "chunk_type": meta["chunk_type"],
            "code_snippet": doc[:400],
            "similarity_score": similarity,
            "line_start": meta.get("line_start", 0),
            "in_stack_trace": file_name in stack_files,
            "reason": "Found in stack trace + semantic match" if file_name in stack_files else "High semantic similarity to error"
        })

    # Sort by score
    ranked.sort(key=lambda x: x["similarity_score"], reverse=True)
    return {"results": ranked[:7], "stack_files_found": stack_files}
```

### Sandbox runner functions (`backend/services/sandbox_runner.py`):

```python
import subprocess, os, shutil
from git import Repo
from openai import OpenAI
import diff_match_patch as dmp_module

openai_client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

def clone_repo(repo_url: str, project_id: str) -> str:
    sandbox_path = f"{os.getenv('SANDBOX_DIR', '/tmp/codelens_sandbox')}/{project_id}"
    if os.path.exists(sandbox_path):
        shutil.rmtree(sandbox_path)
    Repo.clone_from(repo_url, sandbox_path, depth=1)
    # Install dependencies
    if os.path.exists(f"{sandbox_path}/package.json"):
        subprocess.run(["npm", "install", "--legacy-peer-deps"], cwd=sandbox_path, timeout=60)
    elif os.path.exists(f"{sandbox_path}/requirements.txt"):
        subprocess.run(["pip", "install", "-r", "requirements.txt"], cwd=sandbox_path, timeout=60)
    return sandbox_path

def run_command_in_sandbox(sandbox_path: str, command: str) -> dict:
    try:
        result = subprocess.run(
            command.split(),
            cwd=sandbox_path,
            capture_output=True,
            text=True,
            timeout=30
        )
        if result.returncode == 0:
            return {"success": True, "output": result.stdout}
        else:
            error = result.stderr or result.stdout
            return {"success": False, "error": error[:3000]}
    except subprocess.TimeoutExpired:
        return {"success": False, "error": "Command timed out after 30 seconds"}

def generate_fix_with_gpt(sandbox_path: str, target_file: str, error: str) -> dict:
    file_full_path = os.path.join(sandbox_path, target_file)
    with open(file_full_path, "r") as f:
        original_code = f.read()

    response = openai_client.chat.completions.create(
        model="gpt-4o",
        messages=[
            {
                "role": "system",
                "content": (
                    "You are an expert software debugger. "
                    "You will be given a file and an error. "
                    "Return ONLY the complete corrected file content with no explanation, "
                    "no markdown fences, no backticks. Just the raw code."
                )
            },
            {
                "role": "user",
                "content": f"File: {target_file}\n\nCurrent code:\n{original_code}\n\nError:\n{error}\n\nFix the bug."
            }
        ],
        temperature=0.1
    )
    fixed_code = response.choices[0].message.content.strip()

    # Generate diff
    dmp = dmp_module.diff_match_patch()
    diffs = dmp.diff_main(original_code, fixed_code)
    dmp.diff_cleanupSemantic(diffs)

    # Simple summary prompt
    summary_response = openai_client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[{"role": "user", "content": f"In one sentence, what bug was fixed?\n\nDiff:\n{str(diffs[:5])}"}]
    )
    summary = summary_response.choices[0].message.content.strip()

    return {"code": fixed_code, "diff": str(diffs), "summary": summary, "original": original_code}

def apply_fix_to_file(sandbox_path: str, target_file: str, fixed_code: str):
    file_full_path = os.path.join(sandbox_path, target_file)
    with open(file_full_path, "w") as f:
        f.write(fixed_code)
```

## MODULE 7: CHAT WITH CODEBASE (`backend/routes/chat.py`)

```python
from fastapi import APIRouter
from pydantic import BaseModel
from services.embedder import embed_single
from services.chroma_client import query_similar
from openai import AsyncOpenAI
import os

router = APIRouter()
client = AsyncOpenAI(api_key=os.getenv("OPENAI_API_KEY"))

class ChatRequest(BaseModel):
    project_id: str
    message: str
    history: list[dict] = []   # [{ role: "user"|"assistant", content: "..." }]

@router.post("/chat")
async def chat_with_codebase(req: ChatRequest):
    # 1. Embed the user message
    query_embedding = await embed_single(req.message)

    # 2. Find relevant code chunks
    results = query_similar(req.project_id, query_embedding, n_results=6)
    chunks = results["documents"][0]
    metas = results["metadatas"][0]

    # 3. Build context string
    context = "\n\n".join([
        f"### File: {m['file_path']} | {m['chunk_type']}: {m['chunk_name']}\n{doc}"
        for doc, m in zip(chunks, metas)
    ])

    # 4. Call GPT-4o with RAG
    messages = [
        {
            "role": "system",
            "content": f"""You are a codebase expert assistant. Answer the developer's question
using ONLY the code context below. Be specific — mention exact file paths, function names,
and line numbers when possible. Keep answers concise.

CODEBASE CONTEXT:
{context}"""
        },
        *req.history[-6:],   # Last 6 messages for context window efficiency
        {"role": "user", "content": req.message}
    ]

    response = await client.chat.completions.create(
        model="gpt-4o",
        messages=messages,
        max_tokens=800,
        temperature=0.2
    )

    sources = [{"file_path": m["file_path"], "chunk_name": m["chunk_name"]} for m in metas[:3]]
    return {
        "answer": response.choices[0].message.content,
        "sources": sources
    }
```

## DAGRE AUTO-LAYOUT UTILITY (`frontend/src/utils/layout.js`)

```javascript
import dagre from 'dagre'

const NODE_WIDTH = 180
const NODE_HEIGHT = 80

export function applyDagreLayout(nodes, edges, direction = 'TB') {
  const dagreGraph = new dagre.graphlib.Graph()
  dagreGraph.setDefaultEdgeLabel(() => ({}))
  dagreGraph.setGraph({ rankdir: direction, nodesep: 60, ranksep: 80 })

  nodes.forEach(node => dagreGraph.setNode(node.id, { width: NODE_WIDTH, height: NODE_HEIGHT }))
  edges.forEach(edge => dagreGraph.setEdge(edge.source, edge.target))

  dagre.layout(dagreGraph)

  return nodes.map(node => {
    const pos = dagreGraph.node(node.id)
    return { ...node, position: { x: pos.x - NODE_WIDTH / 2, y: pos.y - NODE_HEIGHT / 2 } }
  })
}
```

## API POLLING / SSE HOOK (`frontend/src/hooks/useSSE.js`)

```javascript
import { useEffect } from 'react'
import useProjectStore from './useProjectStore'
import axios from 'axios'

const API = import.meta.env.VITE_API_URL

/**
 * Poll status endpoint every 2 seconds until project is complete.
 * Updates Zustand store with progress.
 */
export function useStatusPolling(projectId) {
  const { setStatus, setGraph } = useProjectStore()

  useEffect(() => {
    if (!projectId) return
    let cancelled = false

    const poll = async () => {
      while (!cancelled) {
        try {
          const { data } = await axios.get(`${API}/api/status/${projectId}`)
          setStatus(data.status, data.progress)

          if (data.status === 'complete') {
            // Load the dependency graph
            const { data: graph } = await axios.get(`${API}/api/map/${projectId}`)
            setGraph(graph.nodes, graph.edges)
            break
          }
        } catch (e) {
          console.error('Status poll failed:', e)
        }
        await new Promise(r => setTimeout(r, 2000))  // Wait 2s between polls
      }
    }

    poll()
    return () => { cancelled = true }
  }, [projectId])
}
```
